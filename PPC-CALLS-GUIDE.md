## 📞 النوع الأول: إعلان اتصال مباشر (Call Ads)

### ✅ المميزات:
| الميزة | الشرح |
|--------|--------|
| **أسرع** | العميل يتصل فوراً |
| **أسهل** | بدون ما يزور الموقع |
| **أرخص** | تكلفة أقل |
| **مباشر** | تواصل فوري |

### ⚠️ العيوب:
| العيب | الشرح |
|-------|--------|
| **مش بتعرف العميل** | مش هتعرف اسمه ولا إيميله |
| **مش بتقدر تتابع** | صعب تعرف إيه الإعلان اللي جاب المكالمة |
| **ممكن مكالمة قصيرة** | ممكن يقفل فوراً |

### 📋 إعداد إعلان الاتصال:

| الخطوة | التفاصيل |
|--------|----------|
| **1** | اذهب لـ Google Ads → حملة جديدة |
| **2** | اختار **Call Ads** (إعلانات الاتصال) |
| **3** | أدخل رقم التليفون: `77175355` |
| **4** | اكتب عنوان الإعلان: "شركة تنظيف قطر | درب المها ☎ 77175355" |
| **5** | اكتب الوصف: "تنظيف فلل، شقق، مجالس. خدمة 24/7. اتصل الآن!" |
| **6** | اختار الكلمات المفتاحية: "شركة تنظيف قطر", "تنظيف فلل" |
| **7** | حدد الميزانية: 50-100 ريال/يوم |
| **8** | فعّل **Call Reporting** عشان تعرف المكالمات |

### 📝 مثال إعلان اتصال:

```
┌───────────────────────────────────┐
│ 📢 إعلان                          │
│                                   │
│ شركة تنظيف قطر                    │
│ درب المها للتنظيف                 │
│                                   │
│ تنظيف فلل، شقق، مجالس             │
│ خدمة 24/7 - خبرة 10 سنوات         │
│                                   │
│ 📞 اتصل الآن: 77175355            │
└───────────────────────────────────┘
```

## 🌐 النوع الثاني: إعلان زيارة الموقع

### ✅ المميزات:
| الميزة | الشرح |
|--------|--------|
| **بتعرف العميل** | بتعرف اسمه، إيميله، سلوكه |
| **بتقدر تتابع** | Conversion Tracking شغال |
| **العميل بيقتنع أكثر** | بيشوف الموقع، الخدمات، التقييمات |
| **بتقدر تعمل Remarketing** | ترجع له بإعلانات تانية |

### ⚠️ العيوب:
| العيب | الشرح |
|-------|--------|
| **أبطأ** | العميل محتاج يدخل الموقع الأول |
| **ممكن يطلع** | يدخل ويخرج من غير ما يتصل |
| **تكلفة أعلى** | تكلفة النقرة أعلى |

### 📋 إعداد إعلان الموقع:

| الخطوة | التفاصيل |
|--------|----------|
| **1** | اذهب لـ Google Ads → حملة جديدة |
| **2** | اختار **Search Ads** (إعلانات البحث) |
| **3** | اكتب عنوان: "شركة تنظيف قطر ☎ 77175355" |
| **4** | اكتب وصف: "تنظيف فلل، شقق، مجالس بالبخار. خدمة 24/7 في كل قطر" |
| **5** | الرابط: `https://darbalmaha.com/` |
| **6** | اختار الكلمات المفتاحية |
| **7** | فعّل **Conversion Tracking** |
| **8** | حدد الميزانية: 50-100 ريال/يوم |
