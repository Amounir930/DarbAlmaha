## مقدمة عن PPC

### إيه هو PPC؟

**PPC = Pay Per Click** (الدفع لكل نقرة)

| المفهوم | الشرح |
|---------|--------|
| **الفكرة** | أنت بتدفع لكل واحد يضغط على إعلانك |
| **المكان** | إعلانات تظهر في أول نتائج البحث |
| **الميزة** | نتائج فورية - مش زي SEO اللي بياخد شهور |
| **التحكم** | أنت بتحدد الميزانية والكلمات والجمهور |

### مثال عملي:

```
عميل بيبحث في Google: "شركة تنظيف قطر"

النتيجة:
┌─────────────────────────────────────┐
│ 📢 إعلان - درب المها للتنظيف        │ ← أنت بتدفع هنا
│ شركة تنظيف قطر الأولى ☎ 77175355   │   لو العميل ضغط
│ خدمات تنظيف فلل، شقق، مجالس         │
│ درب المها - اتصال الآن              │
└─────────────────────────────────────┘

↓ نتائج العضوية (مجانية) ↓
1. شركة تنظيف ...
2. ...
```

```
   ✅ WhatsApp click (نقرة واتساب)
   ✅ Page view (زيارة صفحة)

```

| **ROAS** | > 3x | العائد على الإنفاق |
| **عدد المكالمات** | 30+/شهر | من Conversion Tracking |
| **عدد رسائل واتساب** | 50+/شهر | من Conversion Tracking |

### مثال حسابي:

```
لو صرفت 3,000 ريال في الشهر:
- عدد النقرات: 3,000 ÷ 2 ريال (متوسط CPC) = 1,500 نقرة
- عدد التحويلات: 1,500 × 5% (Conversion Rate) = 75 تحويل
- تكلفة العميل: 3,000 ÷ 75 = 40 ريال
- لو كل عميل بيدفع 300 ريال:
  الإيراد = 75 × 300 = 22,500 ريال
  الربح = 22,500 - 3,000 = 19,500 ريال
  ROAS = 22,500 ÷ 3,000 = 7.5x
```
